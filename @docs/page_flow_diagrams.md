## 1. 核心頁面關係圖

```

|                |     |                   |     |                |
| 登入/註冊頁    +---->+ 主頁/儀表板       +<--->+ 今日任務頁     |
|                |     |                   |     |                |
+----------------+     +--------+----------+     +--------+-------+

                                |                         |
                                v                         v
+----------------+     +--------+----------+     +--------+-------+
|                |     |                   |     |                |

|                |     |                   |     |                |
+--------+-------+     +--------+----------+     +--------+-------+
         |                      |                         |

|                |     |                   |     |                |
| 所有任務頁     +<--->+ 任務詳情頁/側邊欄  +<--->+ 已完成任務頁   |

```

## 6. 關鍵互動模式

### 6.1 拖放操作

- 任務列表內拖放：調整任務優先級/順序
- 跨清單拖放：將任務移動至其他清單
- 日曆視圖拖放：調整任務日期

### 6.2 滑動操作 (移動端)

- 左滑任務：顯示快速操作選項 (完成、刪除、移動、設為重要)
- 右滑任務：標記為完成
- 下拉頁面頂部：重新整理內容

### 6.3 懸停操作 (桌面)

- 懸停任務：顯示快速操作按鈕
- 懸停清單：顯示清單操作選項
- 懸停日期：顯示日期任務預覽

以上流程圖展示了智能待辦事項應用的主要頁面關係與用戶流程，為實現直觀且高效的用戶體驗提供了基礎。設計時應確保流程的清晰性與一致性，最大化用戶任務管理的效率。
